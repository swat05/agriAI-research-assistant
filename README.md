# 🌾 AgriAI Research Assistant

An AI-powered assistant for plant disease detection research.

## Features
- Research literature reviews
- PyTorch / YOLOv8 code generation
- Academic paper writing
- Teaching materials

## Tech Stack
- Streamlit + Google Gemini 1.5 Flash (FREE)
- Domain: Rice Disease Detection (CNN, ViT, YOLOv8)

## Run Locally
pip install -r requirements.txt
streamlit run app.py
